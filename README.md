\# AI Prompt Engineering Playground



This repository contains a collection of small, focused examples created and refined with the help of AI models.  

The goal is \*\*prompt engineering and clear structure\*\*, not “vibe coding”.



Each example lives in its own folder under `site/examples/...` and is published via GitHub Pages.



\## Live demo



GitHub Pages: <https://raven2cz.github.io/ai/>



\- Analog Clock – `site/examples/analog-clock/`  

&nbsp; Smooth animated analog clock with a black frame, classic black hands and a red second hand.
